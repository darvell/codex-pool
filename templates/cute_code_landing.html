codex pool + cute-code

cute-code gets the whole toy box.

Install cute-code, attach it to this pool, and get GPT/Codex models, Claude, Kimi, MiniMax, GLM, remote compaction, web sessions, and the extra runtime pieces cute-code ships with.

GPT/Codex models

Configured through the pool as OpenAI-style models, with WebSocket/SSE Responses support.

Claude + friends

Claude, Kimi, MiniMax, and GLM are registered as Anthropic-compatible custom models.

One settings file

The setup writes cute-code-compatible keys into ~/.claude/settings.json.
